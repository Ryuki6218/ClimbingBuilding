<h2>・ファイルを一式ダウンロードしましょう</h2>
ファイルが一つでも欠けていたら遊べません.
<h2>・ファイルの中の「building-climber.html」を開きましょう</h2>
このhtmlファイルで、ブラウザ上で遊ぶことが可能です.
<h2>・一応ローカルサーバ対応</h2>
ファイルの中のstart_server.batをターミナル上で開いたまま,各接続デバイスの検索エンジン(Chrome等)で「http://(自分のPCに割り振られたIPv4アドレス):8000/」とリンク入力し,その中の「building-climber.html」を開くとスマホ,モバイル上で遊ぶことが出来ます※iOSは試していないので出来ないかもしれません.
